## FAQ

- **Q: F-suffix means no iGPU — what happens if my discrete GPU dies?**
  A: You won't see a display until the GPU is replaced. If this is a concern, choose 7600X (has iGPU) or 5600GT (AM4 with iGPU). For most users with reliable GPUs, this isn't a real issue.

- **Q: Why is 7500F cheaper than 7600X — what's missing?**
  A: Slightly lower clocks (3.7 vs 4.7 GHz base) and no iGPU. Real-world gaming difference with the same discrete GPU is ~5%. The F's value comes from the iGPU you weren't going to use anyway being removed from the silicon cost.

- **Q: Does AM5 require DDR5? Can I use my old DDR4?**
  A: AM5 requires DDR5. No AM5 motherboards support DDR4. If you have a DDR4 kit, you need to either stay on AM4 or sell the DDR4 to fund DDR5.

- **Q: What cooler do you recommend?**
  A: A value-tier tower cooler (Thermalright Peerless Assassin 120 SE, ID-COOLING SE-224-XTS) handles 65W TDP with low noise. AIO is overkill at this TDP.

- **Q: Which motherboard for first AM5 build?**
  A: B650 is the sweet spot — supports DDR5, PCIe 5.0 GPU slot, modest VRM sufficient for 7500F's 65W. ASUS Prime B650-PLUS or Gigabyte B650M AORUS Pro are common entry-tier ExtremePC pairings.

- **Q: First boot taking forever — is this normal?**
  A: Yes. AM5 boards perform DDR5 memory training on first boot which can take 1–3 minutes (sometimes longer). Subsequent boots are normal. If first boot takes 5+ minutes without screen output, check RAM seating.

- **Q: Stock at Onehunga is only 1 unit — is more coming?**
  A: Budget AM5 chips can be supply-limited. If sold out, ETA is typically 1–2 weeks from distributor. Email ExtremePC services to confirm.

- **Q: Can I pick up today from Auckland?**
