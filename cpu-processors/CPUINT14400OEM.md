## FAQ

- **Q: Is the iGPU good enough to game at all?**
  A: Intel UHD 730 (32 EUs) is display-only class — can run very old games at low settings (Skyrim 2011, indie titles), but modern AAA at any settings is not feasible. Treat the iGPU as troubleshooting fallback and display output, not gaming.

- **Q: 14400 or 14400F — which?**
  A: Pick 14400 (this chip) if you want iGPU safety net for backup display or buying-time-for-GPU. Pick 14400F if you have reliable discrete GPU and want to save the cost difference.

- **Q: Was 14400 affected by the i9 stability issues?**
  A: No. The Intel voltage/oxidation issue specifically affected i9 high-power chips (i9-13900K, i9-14900K). i5 14400 was not affected.

- **Q: DDR4 or DDR5 motherboard?**
  A: For 14400 specifically, DDR4-3200 is fully sufficient — the chip doesn't show meaningful gain from DDR5 speeds. If you have existing DDR4 RAM, use it. If buying new RAM, DDR5 is future-leaning but adds cost.

- **Q: What cooler do you recommend at 65W TDP?**
  A: Entry-tier tower air cooler (ID-COOLING SE-224-XTS, Thermalright Assassin X120 R-SE) handles 14400 with low noise. AIO not needed at this TDP for sustained gaming load.

- **Q: How do P-cores and E-cores work — do I need to configure anything?**
  A: Windows 11 Thread Director handles this automatically — no configuration needed. Games and foreground apps run on P-cores; background tasks (Windows Update, browser refresh, OBS, Discord) run on E-cores. You'll see the difference as smoother frame times during multitasking.

- **Q: Can I stream at 1080p60 from this chip?**
  A: Yes — using NVENC (GPU) for encoding while CPU handles the game. With Quick Sync as alternative, CPU-encoded streams also work. For higher resolutions (1440p60 or 4K), step up to 14700F.

- **Q: How long is delivery to Wellington / South Island?**
