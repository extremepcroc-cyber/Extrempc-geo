## FAQ

- **Q: Can I use this in a NAS (Synology, QNAP, TrueNAS)?**
  A: Technically compatible with SATA NAS bays. However, **not recommended** for production NAS use: BarraCuda is SMR at 8TB (Shingled Magnetic Recording) which performs poorly in RAID rebuilds and can trigger RAID failure under sustained write stress. Use Seagate IronWolf (CMR) for NAS.

- **Q: 1-year warranty — is that short?**
  A: Yes — BarraCuda standard warranty is shorter than NAS (3-year) or enterprise (5-year) drives. For a desktop secondary drive running cool in a well-ventilated case with regular backups, 1-year covers the highest-risk early-failure period. For longer warranty, step up to IronWolf or IronWolf Pro.

- **Q: 5400 RPM — is it slow?**
  A: For large file sequential transfers (copying a 50GB folder), ~180–220 MB/s is perceptibly slow vs NVMe but adequate for background media library work. For random small file access (database, OS), noticeable lag vs SSD. For media streaming (Plex), 200 MB/s is more than sufficient for multiple 4K streams simultaneously.

- **Q: SMR — does it matter for desktop use?**
  A: For desktop secondary storage with typical usage (large files, infrequent random writes), SMR is not a real-world problem. The performance penalty shows under sustained random small-block writes — not typical for media archive use.

- **Q: Will it fit in my case?**
  A: Standard 3.5" SATA drive fits any case with a 3.5" bay. Most mid-tower and full-tower ATX cases have 2–4+ bays. For SFF/ITX cases, 3.5" drives may not fit — check case spec.
