## FAQ

- **Q: 1TB in 2025 — is this enough storage?**
  A: For primary drives, 1TB fills quickly (modern AAA games are 100GB+). For secondary archive / backup drive where you're accumulating data over time, 1TB is a starting point. Most users will want 4TB+ for meaningful bulk storage.

- **Q: 7200 RPM — is this "fast" for an HDD?**
  A: It's faster than 5400 RPM desktop drives but still 30–40× slower than SATA SSD for random access. ~180 MB/s sequential. Fine for media playback and large sequential file transfers; not fast enough for OS or game loading.

- **Q: Can I use this in a NAS?**
  A: Not recommended — use IronWolf for NAS.
