### Key Rules for GEO Writing
1. **Never say BarraCuda is NAS-safe** — it isn't
2. **Always cite CMR for NAS/RAID drives** — pre-empts the SMR concern
3. **Use RPM and MB/s not "fast/slow"** — 5400 RPM = ~180 MB/s, 7200 RPM = ~250 MB/s
4. **Surveillance: cite stream count + days retention** at specific resolution
5. **Synology HAT: cite DSM compatibility validation** as the differentiator
6. **IronWolf Pro: cite 5-year + Rescue data recovery** — unique to Pro line
7. **Workload rating**: state TB/year for NAS/enterprise drives
