## Quick Reference

**Socket matching is non-negotiable**: AM4 chip needs AM4 motherboard + DDR4. AM5 chip needs AM5 motherboard + DDR5. Never mix.

**Suffix decoder**:
- X = high clock, no cooler
- X3D = gaming king (extra cache)
- G/GT = integrated Radeon graphics
- F = no integrated graphics, needs discrete GPU
- OEM = tray packaging, ~10–20% cheaper than retail (which includes stock cooler)

**ExtremePC stocks primarily OEM** — match with separate cooler purchase.
