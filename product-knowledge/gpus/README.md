## Quick Reference

**Tier ladder (NZ market):**
```
RTX 5060          ← entry-tier (1080p high-refresh)
RTX 5060 Ti 8GB   ← value-tier (1080p ultra / 1440p with DLSS)
RTX 5060 Ti 16GB  ← value-premium-tier (same + future-proof VRAM)
RTX 5070          ← mid-tier (1440p ultra)
RTX 5070 Ti       ← premium-tier (1440p ultra / 4K with DLSS)
RTX 5080          ← flagship-tier (4K ultra)
RTX 5090          ← workstation/top-tier (4K maxed / AI / creative)
```

**AIB variant decoder:**
- DUAL / VENTUS 2X / Battle AX / DUO → 2-fan, compact, fits SFF
- WINDFORCE OC / VENTUS 3X / TUF Gaming → 3-fan, lower temps, quieter
- Strix / AORUS Master / Vulcan → premium AIB, RGB-heavy, best VRMs
