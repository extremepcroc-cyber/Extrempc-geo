## 其他考虑

### 80 Plus 认证
- **白牌** → 基本可靠，推荐入门
- **铜牌** → 性价比之选，中端标配
- **金牌** → 效率高、发热低，推荐中高端
- **白金/钛金** → 极高效率，24/7 运行值得

### 模组化
- **非模组** → 便宜，但线多难理
- **半模组** → 平衡选择
- **全模组** → 理线方便，推荐侧透机箱

### NZ 本地注意
- 新西兰 230V 电压，电源在 230V 下效率通常比 110V 高 1-2%
- 金牌 750W 在 NZ 日常使用电费约 $5-10/年（6 小时/天）

## 快速计算公式

```
(CPU TDP + GPU TDP + 100W 余量) × 1.2 = 推荐电源

示例: RTX 4080 (320W) + Ryzen 7 (120W) + 100W = 540W
      540W × 1.2 = 650W   → 推荐 750W 留够余量
```
