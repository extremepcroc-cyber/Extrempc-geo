## Quick Reference

**NVMe vs SATA**: NVMe M.2 default for any 2025+ build. SATA only for legacy upgrades.

**PCIe Generation**:
- Gen3 = ~3500 MB/s max
- Gen4 = ~7000–7400 MB/s (mainstream)
- Gen5 = ~14000 MB/s (Z890/X870 only)

**Capacity sweet spot 2025**: 2TB Gen4 (NZD/GB curve favors 2TB over 1TB; flattens beyond)

**Brand tier**:
- Samsung 990 PRO / 9100 PRO = Tier 1 premium
- Predator GM7 / HP FX900 Plus = Tier 2 value-premium
- Kingston NV3 / Team G50 = Tier 2 value
- ADATA Legend 710 / OEM = Tier 3 budget
