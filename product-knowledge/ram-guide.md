# RAM 选购指南 — 容量、频率、时序、兼容性

> 更新日期: 2026-07-15

---

## 容量推荐

| 用途 | 容量 | 说明 |
|:-----|:----:|:-----|
| 办公/上网/看视频 | 8GB | 够用，但建议 16GB |
| 轻度游戏 + 日常 | 16GB | 当前最主流，性价比最高 |
| 主流游戏 + 直播/录屏 | 32GB | 越来越多游戏推荐 32GB |
| 专业工作（剪辑/3D/VM） | 64GB+ | 看具体软件需求 |

### 2026 年建议
- **新装机最少 16GB**，预算允许直接上 32GB
- 32GB 已成为中高端游戏标配（黑神话、城市天际线 2 等吃内存游戏）
- 16GB × 2 双通道 > 32GB 单条

## DDR4 vs DDR5

| | DDR4 | DDR5 |
|:--|:-----|:-----|
| 频率范围 | 3200-3600MHz（主流） | 4800-8000MHz |
| 时序 | CL16-18（较低） | CL30-40（较高） |
| 带宽 | ~25 GB/s | ~40-60 GB/s |
| 价格 | 便宜 | 稍贵 |
| 适合平台 | B660/H610/DDR4 板 | B760/Z790/AM5 |

### 选哪个？

- **AM5 平台（Ryzen 7000/9000）** → 必须 DDR5
- **Intel 12/13/14 代** → 可选 DDR4 或 DDR5，看主板
- **纯游戏** → DDR5 6000 CL30 是甜点，性价比最高
- **预算有限** → DDR4 3200 CL16 仍然够用

## 频率与时序 — Ryzen 特供版

### AMD Ryzen（特别敏感）

| CPU 代数 | 推荐频率 | 说明 |
|:---------|:---------|:-----|
| Ryzen 7000/9000 (AM5) | **DDR5 6000 CL30** | 最甜点，超过 6000 可能不稳 |
| Ryzen 5000 (AM4) | DDR4 3600 CL16 | 1:1 模式最佳频率 |
| Ryzen 3000 (AM4) | DDR4 3200-3600 | 3600 开始可能不稳 |

### Intel

| CPU 代数 | 推荐频率 | 说明 |
|:---------|:---------|:-----|
| 12/13/14 代 | DDR5 5600-6400 或 DDR4 3200-3600 | 相对不敏感 |
| Ultra 200 系列 | DDR5 6400+ | 新架构受益于高频 |

### 时序参考

```
DDR4 3200 CL16 ≈ DDR4 3600 CL18（性能接近）
DDR5 4800 CL40 ← 慢，入门级
DDR5 6000 CL30 ← 甜点（Ryzen 最佳）
DDR5 6400 CL32 ← Intel 高频推荐
DDR5 7200+ CL34 ← 发烧级
```

---

## 双通道 vs 单通道

- **一定要双通道** — 2 条内存比 1 条同容量快 10-30%
- 2×16GB > 1×32GB
- 4 条插满可能降频，优先 2 条

---

## RGB 要不要

- 性能无影响
- 纯外观选择
- 侧透机箱建议 RGB（好看），无侧透随便
