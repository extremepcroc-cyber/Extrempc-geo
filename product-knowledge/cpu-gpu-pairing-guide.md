# CPU-GPU 搭配指南 — 瓶颈分析

> 用途: 让 EVA 回答"这个 CPU 会不会拖累这个显卡？"类问题
> 原则: 分辨率越高，GPU 瓶颈越明显，CPU 瓶颈越不明显

---

## 核心原则

```
1080p → CPU 压力大，更容易 CPU 瓶颈
1440p → GPU 压力适中，大部分组合均衡
4K    → GPU 压力最大，CPU 几乎不会是瓶颈
```

判断 bottleneck 时**先问分辨率**，否则没有意义。

## 加分辦率詳解

### 1080p 高刷新率（最需要 CPU）

```
RTX 4090 + Ryzen 5 5600
  → 1080p 下 CPU 瓶颈明显，5600 喂不饱 4090，帧数可能还不如 4080 + 7800X3D
  → 实际表现: 显卡占用率只有 60-70%

RTX 4070 + Ryzen 7 7800X3D  
  → 1080p 下 GPU 先到瓶颈，CPU 有余力
  → 实际表现: 显卡占用率 95%+，合理搭配
```

**1080p 结论：CPU 比 GPU 更重要。**

### 1440p（甜点分辨率）

```
RTX 4070 + Ryzen 5 5600
  → 1440p 下 GPU 是主要瓶颈，CPU 影响较小
  → 大部分 3A 游戏 GPU 满载，CPU 占用 60-80%

RTX 4080 + Ryzen 5 7500F
  → 1440p 下大部分游戏仍由 GPU 主导
  → 少数 CPU 密集游戏（CS2、瓦罗兰特、Cities Skylines）会有 CPU 瓶颈
```

**1440p 结论：GPU 主导，CPU 没那么关键，但别用太老的 U。**

### 4K（GPU 主导）

```
RTX 4090 + Ryzen 5 7500F
  → 4K 下几乎全是 GPU 瓶颈，CPU 差异很小
  → 7500F 和 7800X3D 在 4K 下帧数差距通常在 5% 以内

RTX 4060 + Ryzen 7 9800X3D
  → 4K 下 GPU 完全跑不满，CPU 再强也没用
  → 属于明显浪费钱的不合理搭配
```

**4K 结论：GPU 决定一切，把钱花在显卡上。**

## 常见误区

| 误区 | 正解 |
|:----|:-----|
| "i9 不会瓶颈" | i9 在 1080p 低画质下同样可能是瓶颈 |
| "4090 一定要配 9800X3D" | 4K 下 7500F 带 4090 差距不到 5% |
| "CPU 越强帧数越高" | 只在一定范围内成立，GPU 先到瓶颈就没区别 |
| "瓶颈永远是坏事" | 任何系统都有瓶颈，关键是瓶颈在哪——GPU 瓶颈才是理想状态 |

---

## 什么情况下考虑 CPU 瓶颈

- 玩游戏时 **显卡占用率 < 90%**
- 同样显卡别人跑 144fps，你只有 100fps
- 降低画质帧数没提升（说明 CPU 跑满了）
- CS2、瓦罗兰特、Fortnite、Cities Skylines、Minecraft 这类 CPU 敏感游戏
