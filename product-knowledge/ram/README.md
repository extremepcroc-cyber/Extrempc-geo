## Quick Reference

**Platform → RAM**:
- AM4 → DDR4 only
- AM5 / LGA1851 → DDR5 only
- LGA1700 → DDR4 or DDR5 (motherboard determines)

**Capacity sweet spots**:
- 16GB — entry / office / light gaming
- 32GB — mainstream gaming / productivity
- 64GB — content creation / streaming pro / AI
- 96GB+ — HEDT / professional workstation

**DDR5 speed sweet spots**:
- 5600 MT/s — value tier
- 6000 CL30 EXPO — AMD AM5 sweet spot (Infinity Fabric sync)
- 6400 CL32 — Intel Core Ultra 200 sweet spot
