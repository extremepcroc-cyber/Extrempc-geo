## FAQ

**Q: Can I install a full-size GPU for gaming?**
No — the N3 has a half-height PCIe slot only, suitable for a low-profile NIC, HBA (IT-mode LSI card), or basic display adapter. It is not designed for gaming GPU installation. For NAS + gaming in one box, consider the Jonsbo N4 (M-ATX support, full-height slot).

**Q: What's the maximum CPU cooler height?**
56mm. This rules out most tower coolers. Compatible options include Noctua L9a/L9i, Thermalright AXP90-X47, and similar low-profile units. If you're running a power-efficient NAS CPU (N100, Ryzen Embedded), stock box cooler or a budget low-profile unit will be adequate.

**Q: Does it support hot-swap drives?**
The trays are tool-free for easy removal, but true hot-swap requires a compatible HBA with hot-plug firmware (e.g. LSI 9207-8i in IT mode) and OS support. The case itself doesn't include a hot-swap backplane — that's a separate component.

**Q: Can I run two 3.5" drives plus SSDs for boot?**
Yes — the 2× 2.5" bays handle SSDs for OS/cache (TrueNAS boot drive, ZFS SLOG, or L2ARC). The 8× 3.5" bays are purely for data HDDs.

**Q: What PSU wattage do I need for 8 drives?**
8× 3.5" HDDs at peak draw roughly 6–8W each during spin-up — approximately 60–65W combined surge. A 450W SFX unit is sufficient for an N100 + 8 HDD build. Add headroom for future drives: 500–550W SFX recommended.
