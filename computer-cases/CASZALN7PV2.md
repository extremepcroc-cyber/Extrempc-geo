## FAQ

**Q: What's the V2 update vs original N7 Plus?**
The V2 revision updates panel design, fan specifications, or manufacturing tolerances vs the original N7 Plus. Specific V2 improvements are documented in Zalman's release notes — check the product page for detailed V2 changelog.

**Q: Does 360mm front radiator work with the included fans?**
The 3× included ARGB fans can be installed as front intake fans behind a 360mm radiator. If installing a 360mm AIO, the AIO's own fans typically replace the front intake fans — the included fans can move to top exhaust or supplemental positions.

**Q: Can this fit an RTX 5080?**
RTX 5080 Founders Edition is 336mm — within the 350mm limit. Most RTX 5080 AIBs run 330–360mm; check your specific AIB dimensions. Cards above 350mm (some 3-fan AIBs) will not fit.

**Q: Is there RGB software required?**
The included RGB hub typically offers standalone button control for cycling lighting modes without software. For full software-controlled sync (matching motherboard ARGB), connect to a 5V ARGB header and use the motherboard's RGB software (ASUS Aura, Gigabyte RGB Fusion, etc.).

**Q: What's the difference between N7 Plus V2 and N7 Plus?**
The V2 is an updated revision — improvements over the original without changing the product line. If both are available, always select V2 as the current production version.
