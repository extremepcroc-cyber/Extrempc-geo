## FAQ

- **Q: Gen3 vs Gen4 — does it matter for gaming?**
  A: In current gaming, Gen3 vs Gen4 difference in actual game load times is minimal (typically <2 seconds per load). DirectStorage-enabled games may show more benefit on Gen4 but most current titles don't fully use it. For typical gaming, Gen3 is sufficient.

- **Q: My motherboard has Gen4 M.2 slots — should I get Gen4 instead?**
  A: For Gen4-capable motherboards, Kingston NV3 or Team G50 (Gen4 1TB at modest premium) is the better choice. For Gen3-only motherboards (Intel 10th/11th Gen, AMD Ryzen 3000), Legend 710 matches the platform.

- **Q: DRAM-less — does it matter?**
  A: For typical gaming and OS use, real-world impact is minimal. DRAM-less drives use Host Memory Buffer (HMB) which gives 95% of DRAM-cache performance. For heavy sustained writes or workstation use, DRAM is meaningful; for gaming, DRAM-less is fine.

- **Q: 3-year warranty vs 5-year on premium drives — does it matter?**
  A: NVMe SSD failure rates are low. 3 years is below premium-tier 5 years but reasonable for the value tier. For consumer gaming / typical use, the 3-year coverage is adequate.

- **Q: Will it boot Windows 11?**
  A: Yes — 1TB capacity is comfortable for Windows 11 + Office + browser + 6–8 modern AAA games. The recommended OS drive baseline tier.

- **Q: Heatsink needed?**
  A: Gen3 drives generate less heat than Gen4. Motherboard built-in M.2 heatsink is sufficient; aftermarket isn't necessary for Gen3.

- **Q: How long is delivery to Wellington / South Island?**
  A: 1–3 business days via insured CourierPost from Auckland.
