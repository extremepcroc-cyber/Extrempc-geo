## FAQ

- **Q: 512GB — enough for OS + light gaming?**
  A: Windows 11 + Office + Chrome + a few smaller games fits. Modern AAA games (Call of Duty, Hogwarts Legacy, Starfield) are 100GB+ each — 512GB fills quickly with 3–4 modern games. For serious gaming, 1TB+ is recommended.

- **Q: 7100 MB/s read vs 3800 MB/s write — does the asymmetry matter?**
  A: For typical desktop use (OS boot, game loading, app launches, file reads), read speed dominates. Writes only matter for content creation (video file copying, image batch processing). FX900 Plus's 7100/3800 is balanced for typical gaming + light productivity.

- **Q: DRAM cache vs DRAM-less — when does it matter?**
  A: DRAM cache (this product has it) helps sustained writes, random IOPS, and OS responsiveness under heavy multitasking. DRAM-less drives (NV3, Legend 710) perform similarly at peak sequential reads but slower under sustained load. For primary OS drives, DRAM is meaningful.

- **Q: Will it work in my laptop?**
  A: If your laptop has an M.2 2280 NVMe slot (most 2020+ business and gaming laptops), yes. Some ultrathin laptops have soldered storage with no upgrade option.

- **Q: HP warranty — what's the coverage?**
  A: HP typically provides 5-year warranty on FX900 Plus SSDs. Handled domestically through ExtremePC NZ.

- **Q: Heatsink needed?**
  A: Motherboard built-in M.2 heatsink recommended for Gen4 drives under sustained load. For Auckland summer ambient (25°C+), the heatsink prevents thermal throttling.

- **Q: How long is delivery to Wellington / South Island?**
