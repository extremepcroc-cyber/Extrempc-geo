## FAQ

- **Q: 256GB — will Windows 11 fit?**
  A: Yes. Windows 11 requires ~64GB; remaining ~190GB usable. Sufficient for OS + Office + Chrome + a few small games. For modern AAA gaming libraries, completely inadequate — modern games are 100GB+ each.

- **Q: Why is it OEM / unbranded?**
  A: Unbranded NVMe drives use NAND chips from various suppliers (not single-source vetted), shorter warranty, no brand recognition. Performance is genuinely OK; reliability and warranty length are the trade-offs vs branded alternatives.

- **Q: 1-year warranty vs 3-year / 5-year on branded SSDs — does it matter?**
  A: NVMe SSD failure rates are very low (<1% typically). 1-year warranty mostly affects edge cases or premature failures. For ultra-budget scenarios where the drive is genuinely intended as temporary or low-value, 1-year is acceptable. For primary OS / important data drives, branded with 5-year warranty is recommended.

- **Q: Will it work on my motherboard?**
  A: If your motherboard has an M.2 NVMe slot, yes. **Note**: some older motherboards have M.2 slots that are SATA-only or PCIe x2 (slower) — Build Team can confirm before purchase.

- **Q: Can I use this as a game drive (secondary)?**
  A: Yes — installing as secondary drive (D: in Windows) for game storage works. But 256GB is small for modern games (CoD: Warzone alone is 100GB+). For game storage, 1TB minimum is recommended.
