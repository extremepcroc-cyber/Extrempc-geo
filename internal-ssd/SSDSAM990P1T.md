## FAQ

- **Q: Samsung 990 PRO vs WD Black SN850X — which is better?**
  A: Both are top Tier 1 Gen4 drives with very close performance. Samsung has slightly stronger NZ availability and longer track record; WD SN850X has Sony PlayStation 5 official endorsement. For NZ market priority, Samsung 990 PRO is the safer pick.

- **Q: 7450/6900 MB/s — does this actually feel faster than NV3's 6000/4000?**
  A: In benchmarks, yes. In real-world gaming load times, marginal (<1 second per load typically). Where 990 PRO's speed advantage shows: sustained workloads (large file copy, video editing, content creation) where the DRAM cache and Samsung NAND quality maintain performance under load. For pure gaming, NV3 is sufficient; for mixed productivity / content creation / premium reliability, 990 PRO.

- **Q: 5-year warranty + 600 TBW — what comes first?**
  A: Whichever comes first ends warranty coverage. For typical consumer use (~50 TBW/year heavy gaming + content creation), 600 TBW = 12+ years. The 5-year time limit is the practical cap. Heavy content creation / 24/7 server use can exceed 600 TBW faster.

- **Q: Heatsink — required?**
  A: Strongly recommended. 990 PRO at sustained Gen4 speeds generates meaningful heat. Most current motherboards include built-in M.2 heatsinks; for older or value motherboards without heatsink, aftermarket heatsink (Sabrent, EKWB) is worth the small investment to prevent thermal throttling.

- **Q: Samsung Magician software — what does it do?**
  A: Samsung's free software for monitoring SSD health, firmware updates, performance benchmarks, secure erase. Works with all Samsung consumer SSDs. Not strictly necessary but useful.

- **Q: Will it boot Windows 11?**
  A: Yes — 1TB is comfortable for Windows 11 + Office + browser + 8–10 modern AAA games + working files.
