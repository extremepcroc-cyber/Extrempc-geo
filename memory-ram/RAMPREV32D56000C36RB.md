## FAQ

- **Q: CL34 vs CL36 at 6000 MT/s — does the latency difference matter?**
  A: ~2–5% gaming improvement in cache-sensitive / CPU-bound titles. For competitive esports chasing 1% lows, CL34 (Vesta II Silver alternative) is the step up. For mainstream gaming, CL36 is fine.

- **Q: AMD EXPO + Intel XMP 3.0 — which to enable?**
  A: On AMD AM5: EXPO. On Intel LGA1700-DDR5 / LGA1851: XMP 3.0. Both profiles factory-tested.

- **Q: Will RGB sync with my non-Predator motherboard?**
  A: Yes — compatible with ASUS Aura Sync, MSI Mystic Light, Gigabyte RGB Fusion, ASRock Polychrome SYNC. Unified RGB control.

- **Q: Black aesthetic — what builds does it match?**
  A: Dark-themed gaming builds, red/black RGB themes, blue/black RGB, most popular RGB ecosystem builds. For white / silver themes, see Vesta II CL34 Silver variant.

- **Q: AMD AM5 sweet spot — why 6000?**
  A: AMD Ryzen 7000 / 9000 chips run Infinity Fabric in 1:1 sync with memory up to ~2000 MHz. DDR5-6000 keeps FCLK at 2000 in sync. Beyond 6000, FCLK desyncs which costs latency.

- **Q: Can I overclock beyond 6000 CL36?**
  A: Possibly with manual BIOS tuning. Hand-screened ICs validated for headroom up to 7200 MHz. Start with 6400 EXPO if available, test stability.

- **Q: Will it fit under my CPU air cooler?**
  A: Vesta II RGB is taller (~44mm) due to lightbar. For tall CPU air coolers (Noctua NH-D15, Phantom Spirit), check clearance. For AIO builds, no issue.

- **Q: How long is delivery to Wellington / South Island?**
