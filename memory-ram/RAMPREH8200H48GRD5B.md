## FAQ

- **Q: Will DDR5-8200 actually run at 8200 on my motherboard?**
  A: Only on Intel Z890 motherboards and AMD X870 motherboards. Predator explicitly marks this kit as Z890/X870 only because lower-tier boards (B650, X670, B850, Z790, B860) lack the memory controller capability and PCB signal integrity to stabilize 8200 MT/s reliably. If your motherboard is not Z890/X870, the 7200 Hermes variant is the right kit (same 48GB at same NZD).

- **Q: Does it actually need Z890/X870 — can I try it on X670E or Z790?**
  A: Possible to try but stability is unlikely at advertised 8200 MT/s — modules will default to JEDEC slower speeds. For full XMP 3.0 8200 MT/s, Z890/X870 is required per Predator's compatibility list.

- **Q: 48GB vs 32GB — do I actually need 48GB?**
  A: For pure gaming: 32GB sufficient. For prosumer workflows (streaming + content creation + browser-heavy + Stable Diffusion + multitasking): 48GB is the sweet spot between 32GB constrained and 64GB premium.

- **Q: 8200 vs 7200 — bandwidth difference in real workloads?**
  A: 8200 MT/s = 65.6 GB/s peak bandwidth per channel; 7200 = 57.6 GB/s. ~14% more bandwidth. Real-world workload improvement: 3–7% in memory-bandwidth-sensitive tasks (Core Ultra 200 productivity, AI inference, large dataset processing). For most gaming, marginal. Worth the platform-restriction trade-off if you're already on Z890/X870 and want maximum bandwidth.

- **Q: 10-layer PCB at 8200 MT/s — does this matter?**
  A: Yes — critical. At 8200 MT/s, signal integrity is the key limiting factor. Standard 6-8 layer PCBs have signal interference and ground plane issues that prevent stable operation at this frequency. 10-layer PCB enables 8200 to be a stable everyday operation, not just a benchmark-only setting.

- **Q: Auckland summer heat — is the bundled fan necessary?**
  A: Strongly recommended for sustained 8200 MT/s operation in Auckland 25°C+ ambient. Without the fan in average case airflow, prolonged high-frequency operation can cause thermal throttling. Install the bundled fan for guaranteed stability.

- **Q: Can I overclock beyond 8200?**
  A: Hand-sorted ICs are validated to 8400 MT/s. Manual BIOS tuning may stabilize 8400 on the best-binned silicon with elite cooling and Z890/X870. For competitive overclocking community, headroom exists.
