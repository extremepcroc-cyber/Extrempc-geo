## FAQ

- **Q: RGB vs non-RGB — does RGB affect performance?**
  A: No — RGB is purely aesthetic. Performance identical. Choose by build theme priority.

- **Q: PALLAS II vs Vesta II — what's the difference besides RGB?**
  A: PALLAS II is non-RGB lower-profile heatsink (43.75mm); Vesta II adds RGB lightbar (~50mm taller). PALLAS II has cleaner aesthetic + better CPU air cooler clearance; Vesta II has showcase RGB + slightly more visual presence.

- **Q: AMD EXPO + Intel XMP 3.0 — which to enable?**
  A: On AMD AM5: EXPO. On Intel LGA1700-DDR5 / LGA1851: XMP 3.0. Both profiles factory-tested.

- **Q: Silver aesthetic — what builds does it match?**
  A: White / light-themed cases (Lian Li O11 Snow, NZXT H7 Flow White), silver-accent motherboards (ASUS Prime X670E), Scandi / minimalist aesthetics where RGB is unwanted.

- **Q: AM5 sweet spot — why 6000?**
  A: AMD Ryzen 7000 / 9000 Infinity Fabric (FCLK) syncs 1:1 with memory up to ~2000 MHz. DDR5-6000 keeps FCLK at 2000 in sync — beyond 6000, FCLK desyncs which costs latency.

- **Q: Can I overclock to 6400 / 7200 MT/s?**
  A: Possibly with manual BIOS tuning. Hand-selected ICs validated for headroom. Stability silicon-dependent — start with 6400 if available, test before pushing.

- **Q: Will it fit under my CPU air cooler?**
  A: Yes — PALLAS II at 43.75mm height is lower than RGB Vesta II variants. Fits comfortably under Noctua NH-D15, Thermalright Phantom Spirit 120 EVO, Be Quiet Dark Rock 4.
